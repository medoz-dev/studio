# **App Name**: Chez Maman-Didi Inventory

## Core Features:

- Stock Remaining: Tabulated inventory counts, fetched from local storage, of beverage stock at 'BAR RESTAURANT CHEZ MAMAN-DIDI'.
- Stock Arrival: Calculate stock arrival by units, cases, or packages for each drink type, save arrival logs in local storage and display current data in a UI table
- General Calculation: System to compute beverage sales, cash, calculate totals for each date using stored values in local storage and other calculations such as sum of arrival and previous values

## Style Guidelines:

- Primary color: Dark moderate blue (#2C3E50) - For header, tabs and primary buttons.
- Background color: Light gray (#F5F5F5) - For the overall page background.
- Accent color: Strong orange (#F39C12) - Highlights important elements such as titles and summary box borders.
- Body and headline font: 'PT Sans', a humanist sans-serif. Good for both headers and body.
- Use simple, flat icons to represent different types of inventory and actions.
- Tabbed layout to separate the stock, arrival, and calculation functionalities. Use cards to group related form elements and tables.
- Subtle transitions when switching tabs or updating data in tables. To maintain the user experience keep the animations at a maximum of .3s duration.
